# Data Model

## Datasets

| File                         | Description             |
| ---------------------------- | ----------------------- |
| `DLP_Exceptions_MVP_EN.xlsx` | DLP exception inventory |
| `DLP_Alerts_MVP_EN.xlsx`     | DLP alerts export       |

---

## DLP Exceptions Dataset

Main source for exception governance.

| Column                   | Purpose                                         |
| ------------------------ | ----------------------------------------------- |
| `ExceptionID`            | Unique identifier                               |
| `Requester`              | User requesting the exception                   |
| `Department`             | Requester's business area                       |
| `BusinessJustification`  | Business reason                                 |
| `Channel`                | DLP location/channel                            |
| `DLPPolicy`              | DLP exception policy                            |
| `SensitiveDataType`      | Type of data affected                           |
| `CommunicationStatus`    | Communication status with the user              |
| `FormResponseStatus`     | Form response status                            |
| `Approver`               | Approval owner                                  |
| `ApprovalStatus`         | Approval status                                 |
| `ExpiryDate`             | Expiration date                                 |
| `ExceptionStatus`        | Calculated exception status                     |
| `AlertsLast30Days`       | Related alerts in the last 30 days              |
| `LastActivityDate`       | Last DLP activity                               |
| `ReadyForAssessment`     | Indicates whether the exception can be assessed |
| `RiskScore`              | Risk scoring                                    |
| `RiskLevel`              | Low / Medium / High / Critical                  |
| `RiskExplanation`        | Risk drivers                                    |
| `RecommendedAction`      | Proposed action                                 |
| `LastRiskAssessmentDate` | Assessment date                                 |

---

## DLP Alerts Dataset

Source of real activity generated by DLP rules.

| Column                             | Purpose                          |
| ---------------------------------- | -------------------------------- |
| `File`                             | Affected file                    |
| `Location`                         | Channel where the alert occurred |
| `User`                             | Associated user                  |
| `Occurred`                         | Alert date                       |
| `Sensitive information type`       | Sensitive data detected          |
| `Sensitive information type count` | Number of matches                |
| `Policy`                           | DLP policy                       |
| `Rule`                             | DLP rule                         |
| `Email recipient`                  | Recipient, if applicable         |
| `File extension`                   | File type                        |
| `File size`                        | Size                             |
| `Application`                      | Application involved             |
| `Enforcement mode`                 | Policy enforcement mode          |
| `Destination domain`               | External domain, if applicable   |
| `Activity type`                    | Detected action                  |

---

## Relationship between datasets

| Exceptions  | Alerts   |
| ----------- | -------- |
| `Requester` | `User`   |
| `DLPPolicy` | `Policy` |

`Occurred` is used to calculate activity in the last 30 days.

---

## Fields calculated by the agent

| Field                    | Source           |
| ------------------------ | ---------------- |
| `AlertsLast30Days`       | Alerts export    |
| `LastActivityDate`       | Alerts export    |
| `RiskScore`              | Agent scoring    |
| `RiskLevel`              | Agent scoring    |
| `RiskExplanation`        | Agent assessment |
| `RecommendedAction`      | Agent assessment |
| `LastRiskAssessmentDate` | Analysis date    |
